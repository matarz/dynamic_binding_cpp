This project was a proof of concept for using dynamic binding in c++. The assignment asked that we build a program that would manage at least three devices using one common interface.

Pick three high tech devices that you frequently use. Instead of re-implementing the functionality for common overlapping features, wouldnâ€™t a hierarchy be best to simulate their functionality? The best choices in technology would all have similar functionality, such as: turn on, turn off, run application, play/call, go back, etc. Find the set of common functionality and pull it up to the common base class.
